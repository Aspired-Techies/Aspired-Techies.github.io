Machine-Learning

"Machine learning will automate jobs that most people thought could only be done by people."
~Dave Waters

Defination
Machine learning is the study of computer algorithms that improve automatically through experience and by the use of data.
Machine learning algorithms use historical data as input to predict new output values. Machine learning algorithms learn by experience, similar to how humans do.

Why to learn Machine-Learning ?

  1. Better Career Opportunities and Growth :

    2. Machine learning makes a mockery of anything that can be called “important”.
    It is predicted to grow from to $19.9 billion by the end of 2025, from a mere $1.07 billion in 2016.

  2. Machine learning and Data Science are intricately linked :

    3. Machine Learning is just a shadow of Data Science. To take your career as high as you can’t even imagine, you can become competent in both these fields.
    In many organisations, ML engineers and Data Scientists work together on products, so it is likely that you will be exposed to the Data Scientists’ perspective if you’ve already become an ML engineer.

Learn Core ML Algorithms

  1. Most suitable languages for machine learning are Python and R
    Learn Python Libraries:
    1. Learn Numpy.

  2. Gradient Descent
  3. Slope
  4. Supervised vs Unsupervised learning
  5. Reinforcement Learning
  6. Basic Linear Regression
  7. Working of all such similar models
  8. Clustering
  9. Get Data (webscraping, API calls, image libraries): coding background.
  10. Clean/munge the data. This takes all sorts of forms.
  11. Tune your algorithm and finally Visualize!
